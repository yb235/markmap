## For First-Time Users

If you're new to Markmap, we recommend reading the documentation in the following order:

1. Start with [Overview](./01-overview.md) to understand what Markmap does
2. Read [Getting Started](./07-getting-started.md) to begin using Markmap
3. Check [Examples](./08-examples.md) for common use cases
4. Explore [Architecture](./02-architecture.md) and [Workflow](./04-workflow.md) to understand how it works internally
5. Refer to [APIs](./05-apis.md) when you need to integrate Markmap programmatically
